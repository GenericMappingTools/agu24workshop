Tutorial 3 - scientific Python ecosystem üêç: Xarray (gridded data üåê)#

In this tutorial we will to process and visualize raster data using PyGMT‚Äôs integration with Xarray.

0Ô∏è‚É£ What is Xarray?#

Xarray is an open source project and Python package for working with n-dimensional data using labeled dimensions and coordinates.

1Ô∏è‚É£ Why use Xarray with PyGMT#

Xarray provides a set of useful features for working with multi-dimensional data and unlocks even more functionality through its rich ecosystem. Some specific benefits of integrating Xarray and PyGMT include:

  • Simplifying the extension of PyGMT functionality to 3-D+ datasets

  • Access additional datasets, such as those located on the cloud

  • Use out-of-core computing with libraries like Dask

  • Interactively visualize by using Xarray, PyGMT, and dashboarding tools like Panel and Bokeh

  • Work with both raster and vector data in memory

2Ô∏è‚É£ Getting started#

First we‚Äôll import all the libraries used throughout the tutorial.

import gcsfs
import pygmt
import xarray as xr

Let‚Äôs use a PyGMT function to explore an Xarray DataArray. We can use the pygmt.datasets.load_earth_relief function to load one of GMT‚Äôs remote datasets. This will take longer the first time that it‚Äôs run because GMT downloads the data from the GMT server. We‚Äôll specify the server in order to quicken download speeds to Washington, D.C., USA.

with pygmt.config(GMT_DATA_SERVER="NOAA"):
    grid = pygmt.datasets.load_earth_relief(resolution="01d")

3Ô∏è‚É£ Plotting raster data#

We can pass this dataset to Figure.grdimage to plot the dataset, specifying a projection and colormap. We use the Winkel Tripel projection (‚ÄúR‚Äù) with a 12c width to show the whole world.

fig = pygmt.Figure()
fig.grdimage(grid=grid, projection="R12c", cmap="SCM/oleron", frame=True)
fig.show()

4Ô∏è‚É£ Processing raster data#

All of the PyGMT raster processing functions can accept and return Xarray DataArrays. As one example, we‚Äôll use the grdgradient function to create a hillshade raster. This will calculate the reflection of a light source projecting from west to east at an altitude of 30 degrees.

region = [-119.825, -119.4, 37.6, 37.825]
grid = pygmt.datasets.load_earth_relief(resolution="03s", region=region)

hillshade = pygmt.grdgradient(grid=grid, radiance=[270, 30])

6Ô∏è‚É£ Open data directly from the cloud#

Xarray allows you to open data directly from cloud object storage. This can be useful to avoid downloading the entire dataset, since Xarray can subset to a specific section of interest before loading the data into memory.

Here we show this functionality based on Xarray‚Äôs tutorial. We‚Äôll load a specific dataset from the Coupled Model Intercomparison Project Phase 6 (CMIP6), calculate the sea level change between 2015 and 2100, and plot the results using PyGMT.

First, let‚Äôs load CMIP6 data from Google Cloud Storage

fs = gcsfs.GCSFileSystem(token="anon")

store = fs.get_mapper(
    "gs://cmip6/CMIP6/ScenarioMIP/NOAA-GFDL/GFDL-ESM4/ssp585/r1i1p1f1/Omon/zos/gr/v20180701/"
)
ds = xr.open_zarr(store=store, consolidated=True)
ds

Calculate the different in sea level between a date in 2100 and 2015

zos_2015jan = ds.zos.sel(time="2015-01-16").squeeze()
zos_2100dec = ds.zos.sel(time="2100-12-16").squeeze()
sealevelchange = zos_2100dec - zos_2015jan

Make a plot of the difference in sea level

fig = pygmt.Figure()
pygmt.makecpt(cmap="vik", series=[-1, 1, 0.5], continuous=True)
fig.grdimage(grid=sealevelchange, cmap=True, projection="R15c", frame=True)
fig.colorbar()
fig.show()
